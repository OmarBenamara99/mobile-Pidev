## Overview
Khademni is a multi-platform project connecting users and freelancers. This repository contains the **mobile version** built with **Codename One**, part of the integrated desktop and web ecosystem.

## Tech Stack
- **Mobile Framework:** Codename One  
- **Database:** MySQL (shared across platforms)  
- **Other Platforms:** Desktop (JavaFX), Web (Symfony)

## Key Features
- Connects users seeking services with freelancers efficiently  
- Supports search, filtering, and scheduling via Google Calendar  
- Automates email and SMS notifications  
- Handles complaints and improves navigation for a smooth user experience

## Achievements
- Delivered a functional mobile app integrated with desktop and web platforms  
- Ensured multi-platform consistency via a **shared database**
